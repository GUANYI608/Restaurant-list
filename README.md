# 餐廳清單
此專案為使用者提供餐廳資料的瀏覽與搜尋功能

## 功能列表
1. 使用者可以在首頁看到所有餐廳與它們的簡單資料：
- 餐廳照片
- 餐廳名稱
- 餐廳分類
- 餐廳評分
2. 使用者可以再點進去看餐廳的詳細資訊：
- 類別
- 地址
- 電話
- 描述
- 圖片
3. 使用者可以透過搜尋餐廳名稱來找到特定的餐廳
4. 使用者可以新增餐廳資料
5. 使用者可以修改餐廳的資訊
6. 使用者可以刪除餐廳資料

## 使用工具
- 開發環境：Visual Studio Code
- 執行環境：Node.js
- 應用程式架構：Express
- 模板引擎：Express-Handlebars
- 前端框架：Bootstrap
- 圖示資源：Font Awesome
- 資料庫：MongoDB
- 資料庫圖形介面：Robo 3T
